Instrucciones:
Para usar el código que traduce a lenguaje morse y al viceversa, primero
se ejecuta el archivo Morse_totext.java en el package Morse, se nos 
pedirá teclear M o T dependiendo de lo que queramos hacer, por ultimo
se escribirá el texto en minúsculas que queramos traducir y damos Enter.

Para el código que cambia números decimales a romanos, se debe ejecutar 
el archivo NRomanos.java alojado en el package Romano, se nos pedirá escribir 
un numero en el rango de 1 a 1000, lo escribimos y Enter.

Por último, para el código de herencia, nos dirigimos al archivo Main.java
dentro del package He_Objetos y seguimos cada instrucción acompañada de un 
Enter al final.
